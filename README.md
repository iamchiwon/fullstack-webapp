# Fullstack WebApp Example

Next.js App on Amplify

- Use Next.js frontend/backend API
- With Amplify:
  - Use database
  - Use storage
  - Use function
  - Use authentication

## Steps

1. create next.js app

```bash
$ npx create-next-app@latest
```

2. add amplify

```bash
$ npm create amplify@latest
```
